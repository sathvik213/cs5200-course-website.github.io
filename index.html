Syllabus

This course introduces RDBMS concepts including SQL, relational algebra, ER diagrams, normalization, and more.

Learning Goals

  • Design normalized relational schemas
  • Write SQL queries (DDL, DML, DCL)
  • Understand transactions, indexes, constraints
  • Build DB-backed applications with UI

Reference Books

Introduction to Databases and Data Warehouses, Jukić et al., 2nd Edition, Prospect Press
